📊 Customer Churn Analysis

📌 Overview

This project analyzes customer churn data to identify key factors that cause customers to leave a service. The goal is to understand customer behavior and provide insights to improve retention.

🎯 Key Objectives
* Analyze customer churn patterns
* Identify important features affecting churn
* Perform data cleaning and visualization
* Generate business insights

📊 Key Insights
* Month-to-month contracts have the highest churn
* New customers (low tenure) are more likely to leave
* Higher monthly charges increase churn
* Customers without additional services churn more

🛠️ Tools Used
* Python
* Pandas
* Matplotlib
* Seaborn

Conclusion
This project demonstrates how data analysis can help businesses understand customer behavior and reduce churn through data-driven decisions.
